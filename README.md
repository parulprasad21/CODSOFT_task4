# Basic Calculator

This is a simple calculator web application created using HTML, CSS, and JavaScript.  
It can perform basic arithmetic operations like addition, subtraction, multiplication, and division.

## Live Demo
You can view the live calculator here:  
[Calculator Live Link](https://parulprasad21.github.io/CODSOFT_task4/)

## Features
- Interactive calculator interface
- Addition, subtraction, multiplication, and division
- Clear button to reset the display
- Responsive design for mobile and desktop

## Technologies Used
- HTML
- CSS
- JavaScript
